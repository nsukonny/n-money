Service for control of expenses and income

== Description ==
I develop that service for my self, but you can use it if you need it.

== Installation ==

1. Upload the `nmoney.zip` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add shortcode [nmoney_panel] to needed page
4. Enjoy
